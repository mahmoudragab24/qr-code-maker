# QR Code Generator

A simple Python program to generate QR codes from URLs or text. 
The user can specify a custom filename for each QR code to prevent overwriting previous codes.

## Features
- Input any URL or text
- Choose your own filename for each QR code
- Prevent overwriting existing files
- Easy to use and extendable

## Requirements
- Python 3.x
- qrcode library
- Pillow (automatically installed with `qrcode[pil]`)

Install the required library using pip:

```bash
pip3 install qrcode[pil]
